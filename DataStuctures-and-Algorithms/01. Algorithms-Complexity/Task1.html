Ex. 1) What is the expected running time of the following C# code? Explain why. Assume the array's size is n. 

long Compute(int[] arr)
{
	long count = 0;
	
	for (int i = 0; i < arr.Length; i++)
	{
		int start = 0, end = arr.Length - 1;
		while (start < end)
		{
			if (arr[start] < arr[end])
			{
				start++;
				count++;
			}
			else
			{
				end--;
			}
		}
	}
	return count;
}

Answer:

The outer 'for' loop is iterating n times and on every step the inner 'while' loop is iterating another n times so the answer is 0(n*n)
